# Inventory_Management_System
Python / SQL Database Bookstore management program. Capstone Project.

Project completed. Full CRUD operations enabled. 

Fully manageable Book inventory system that allows user to complete following using menu system:
    1.) Allowing User to enter in new inventory item, match or add new author, and add quantity.
    2.) View full list of inventory.
    3.) Update specific books by updating author, qty, and more
    4.) Deleting books completely from inventory system.
    5.) Do a search of inventory for individual book based on entered parameters.
